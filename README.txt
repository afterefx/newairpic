Create the database structure by using the schema text file provided

    mysql.schema

Add an user to the database and also create a driver page to give you the
encoded password to place in the password field. Be sure and use the backend
provided.

Update the following files:
    class.Settings.php

        You need to specify the full path of the location of your
        airpicStrings.xml file.
        (/var/www/mySite/includes/airpicStrings.xml)

    airpicStrings.xml

        You need to fill out your database infromation and site settings


NOTE: the permissions for the log folder may need to be fixed
